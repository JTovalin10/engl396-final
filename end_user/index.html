Overview

My Favorite Albums is a web application that allows users to explore and analyze one person's album rating and annual ranking data through interactive data visualizations.

Prerequisites: fundamental computer skills to use the app.

The application is organized into six tabs, each providing a different way to filter and view the data. The sections below explain what each tab does and how to interact with it.

Number One Albums

Find the owner's favorite album for each year within a selected range.

This section uses a range slider to aggregate data and show the owner's favorite album for each year within the selected range. At the bottom of the page, a data table displays the owner's favorite album by year.

Input
ControlTypeDescription
Year RangeRange SliderDrag the handles to set the start and end year for the results
Output: data table with the following fields
FieldDescription
YearThe year of the #1 album
AlbumThe top-ranked album for that year
ArtistThe artist who made the album

Top Albums by Year

See all of the owner's ranked albums for one specific year.

This section uses a drop-down menu that allows you to select one specific year, defaulting to 1993. When you choose a year, it displays a data table of the owner's favorite albums for that year and where each album ranks.

Input
ControlTypeDescription
YearDrop-Down MenuSelect the year you want to view rankings for (default: 1993)
Output: data table with the following fields
FieldDescription
RankingThe album's rank for that year
AlbumThe album title
ArtistThe artist who made the album

Artist's Albums

View all albums the owner has listened to for a specific artist, along with their ratings.

This section uses a drop-down menu to select a specific artist and see what albums the owner has listened to, the year they were released, and the rating the owner gives each album. At the bottom of the data table, you will see the number of albums ranked for the selected artist and the average rating.

Input
ControlTypeDescription
ArtistDrop-Down MenuSelect the artist whose albums you want to view
Output: data table with the following fields, plus summary stats below the table
FieldDescription
AlbumThe album title
YearThe release year
RatingThe owner's rating for the album

Favorite Artists

Discover the owner's top-rated artists, filtered by minimum album count.

This section uses a drop-down menu that creates a data table with at most 15 artists, organized by average rating in descending order. You can filter by the minimum number of albums an artist must have, and choose whether to include EPs and live albums.

Input
ControlTypeDescription
Minimum AlbumsDrop-Down MenuSet the minimum number of albums an artist must have to appear in the list
Include EPs / Live AlbumsCheckboxToggle whether EPs and live albums count toward an artist's total
Output: data table (up to 15 rows) with the following fields
FieldDescription
ArtistThe artist name
RatingThe artist's average rating across all their albums
AlbumsThe number of albums included in the rating

Artist Comparison

Compare how two artists' album ratings have changed over time.

This section creates a comparison between two artists that you select. It generates a comparative line chart that displays each artist's average rating of albums released in a given year.

Input
ControlTypeDescription
Artist 1Drop-Down MenuThe first artist to compare (plotted as a red line)
Artist 2Drop-Down MenuThe second artist to compare (plotted as a blue line)
Output: comparative line chart
ElementDescription
Red lineThe first artist's average album rating per year
Blue lineThe second artist's average album rating per year
X-axisYear
Y-axisAverage rating

Appendix: Terminology

Definitions for interface terms used throughout the application.

Range Slider

A user interface component that allows you to select a range of values from a set of numbers.

Handle

The draggable element on a range slider that users interact with to select a value or range of values. It moves along the slider's track and visually indicates the current selected value(s). In a dual-range slider, there are typically two handles — one for the minimum value and one for the maximum value — allowing users to define a specific range.

Drop-Down Menu

A graphical user interface element that allows users to choose from a list of options that appears when they click on or hover over a button or other control.

Comparative Line Chart

A visual representation of data used to compare and contrast two or more data sets.

Aggregate

To filter and combine data based on criteria you select.
